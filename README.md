## **실행 화면**

### 스플래시 및 로그인 화면


### Tab1 - 프로필

* 로그인한 유저의 프로필 사항 기재
* 내가 작성한 글, 내가 좋아요를 누른 글 열람
* 회원 탈퇴



### Tab2 - 채팅


* 소켓을 활용한 실시간 채팅 서비스
* 웹소켓을 사용하여 글, 이미지, 좋아요 아이콘 전송 가능
* 수신한 메세지(글, 이미지)를 클릭/재클릭하면 좋아요 아이콘을 표시



### Tab3 - 커뮤니티


* 전체 게시글 열람
* 내 게시글일 시 수정, 삭제 버튼이 있고 댓글을 삭제할 수 있는 버튼이 보여짐
* 내 게시글이 아닐 경우 댓글 작성만 가능
